# 🌐 Сетевая автоматизация: скрипты для настройки коммутаторов и маршрутизаторов

### Этот репозиторий содержит инструменты для автоматизации рутинных задач сетевых инженеров, включая:

- Массовую настройку ACL на коммутаторах

- Автоматизацию конфигурации D-Link коммутаторов

- Генерацию конфигураций для оборудования Juniper

  - Проверку доступности IP-адресов

- Комплексную автоматизацию отключения портов оборудования по данным CRM

- Комплекс инструментов для мониторинга и диагностики GPON сетей на базе SNMP и PostgreSQL. Оборудование SNR BDCOM.

## 🎯 Зачем это нужно?

### Создание унифицированного, масштабируемого инструментария для автоматизации рутинных задач на оборудование сетевой инфраструктурой, обеспечивающего:

- Снижение ошибок ручного ввода

- Сокращение время специалиста на рутинные задачи
  
- Масштабируемость

- Устранение человеческого фактора

<b>2. Настройка ACL на D-Link-коммутаторах серии 32xx </b>
  </summary>

#### <i> Пример: "Определение access, trunk, uplink и downlink портов. Создать ACL" </i>

- Определяет, какие порты — access
- Находит "uplink"-порты и "downlink" - порты
- В зависимости от модели устройства и версии софта, загружает ACL на основе найденных данных
- Осуществляет проверку ACL листов необходимых для данного типа оборудования

## 🛡 Безопасность и этика

- Никогда не храните пароли в коде — скрипты запрашивают их вводом при запуске.

- Используйте только на устройствах, к которым у вас есть право доступа.

- Тестируйте на стендах перед массовым применением.
