### Этот репозиторий содержит три взаимосвязанных скрипта для автоматизации процесса отключения портов сетевого оборудования на основе данных из CRM-системы. Скрипты последовательно извлекают данные из наряда, обрабатывают их и выполняют необходимые команды на сетевом оборудовании. ###

#### Взаимодействие скриптов


```
main.py (бесконечный цикл)
       │
       ├── script_parsing_crm.py
       │         │
       │         ├── Вход в CRM
       │         ├── Поиск наряда по ID
       │         └── Сохранение данных в .pkl файлы
       │
       └── shutdown_pppoe_port.py
                 │
                 ├── Загрузка данных из .pkl
                 ├── Проверка дат и условий
                 ├── Подключение к оборудованию
                 ├── Определение модели
                 └── Выполнение отключения

```
