📋 Этот репозиторий содержит два взаимосвязанных скрипта для мониторинга и диагностики GPON сетей. Система позволяет автоматически собирать данные об ONU устройствах через SNMP, сохранять их в PostgreSQL и предоставлять удобный инструмент для быстрого поиска информации об абонентах.

🔄 Взаимодействие скриптов

snmp_onu_req.py (Бесконечный цикл мониторинга)

│
├── Асинхронный SNMP опрос OLT
       │         │
       │         ├── Получение данных по OID
       │         ├── Парсинг SNMP Walk
       │         └── Сопоставление метрик
       │
       ├── Обработка и сохранение
       │         │
       │         ├── Расчет времени в состоянии
       │         ├── Сохранение в PostgreSQL (UPSERT)
       │         └── Резервное копирование в файл
       │
       └── Ожидание следующего цикла (POLL_INTERVAL)

onu_status.py (Клиентский поиск ONU)
       │
       ├── Поиск в PostgreSQL
       │         │
       │         ├── По HWTC серийному номеру
       │         └── По имени ONU
       │
       ├── Актуальный SNMP опрос
       │         │
       │         ├── Статус ONU
       │         └── Client LAN Status
       │
       └── Отображение информации
                 │
                 ├── Форматирование данных
                 ├── Цветовая индикация статуса
                 └── Вывод в MessageBox
